# Електронна платформа за управление на складова наличност

## Описание
Този проект е уеб платформа за управление на складова наличност, предназначена за търговски обекти, фирми и учебни организации. Системата осигурява проследяване на артикули, складови операции, история на движението и контрол на потребителския достъп.

## Основни Функционалности
1. Регистрация и вход в системата с потребителски роли администратор и потребител;
2. Управление на складови артикули с възможност за добавяне, редактиране, изтриване и търсене;
3. Складови операции за приемане и изписване на стока;
4. Проследяване и съхраняване на история на всички извършени складови операции;
5. Контрол на наличностите и сигнализиране при достигане на минимални количества;
6. Генериране на справки за текуща наличност и движение на артикули;
7. Изграждане на интуитивен и удобен потребителски интерфейс;
8. Осигуряване на защита и коректност на въведените данни.

## Изисквания
*   **Node.js**
*   **npm**
*   **Браузър с поддръжка на JavaScript**

## Инсталация
Инсталирайте нужните зависимости:
```bash
npm install
```

## Стартиране
Стартирайте фронтенда и backend сървъра:
```bash
npm run dev
```

## Технологичен Стек
*   **React + TypeScript + Vite:** За клиентската част.
*   **Express:** За backend API.
*   **SQLite:** За локално съхранение на данните.
*   **JWT + cookie-parser:** За автентикация и сесии.
*   **Tailwind CSS + shadcn/ui:** За интерфейса.
*   **TanStack Query, Recharts, jsPDF, XLSX:** За заявки, визуализации и отчети.
